def newton_pts(f, df, x, xtol=1e-8, ytol=1e-8, maxiter=100):
    pts = [x]
    iters = 0
    while True:
        q = f(x)/df(x)
        xnext = x - q
        if xnext not in pts: pts.append(xnext); 
        iters +=1
        x = xnext
        if not((abs(f(x))>ytol) and (abs(q)>xtol) and (iters < maxiter)):
            return pts

Student's answer Score: 1.0 / 1.0 (Top)

O erro em x calculado com $|{\frac{f(x)}{df(x)}}|$, isso significa que uma operao entre dois nmeros resultados de outras operaes feitas anteriormente, para determinar f(x) e df(x). Esses dois nmeros j tm um erro, devido limitao de preciso do computador, logo, fazer mais uma operao com eles, produz mais erro. Isso resulta em um erro maior do que o erro real, que seria calculado usando a raiz, se fosse conhecida. Usando o erro em y, s precisamos calcular ${f(x)}$, envolvendo menos contas feitas pelos computador, resultando em um valor mais prximo do valor real de f(x), que em mdulo o erro em y

Questão 3: Polinômios em Python¶

Para no precisar ficar implementando polinmio por polinmio, implemente uma funo cria_poli(coefs) que retorna uma (outra) funo poli(x), que computa o valor do polinmio

$p(X)=$ coefs[0] $+$ coefs[1] $X+$ coefs[2] $X^2+\dots+$ coefs[-1] $X^n$

em $X=$x. As funes de polinmios do numpy (poly1d, polyval, polyder) podem lhe ser teis :-)

No mesmo esprito, crie a funo cria_derivada_poli(coefs), que retorna uma funo derivada_poli(x) que computa a derivada do polinmio acima em $X=$x.

def cria_poli(l):
    def poli(x):
        p = np.poly1d([l[len(l)-1-i] for i in range(len(l))])
        return np.polyval(p,x)
    return poli

def cria_derivada_poli(l):
    p = np.poly1d([l[len(l)-1-i] for i in range(len(l))])
    def derivada_poli(x):
        der_poly = np.polyder(p)
        return np.polyval(der_poly,x)
    return derivada_poli
