Calculando um autovalor qualquer¶

Questão 1: Iteração inversa¶

Podemos calcular um autovalor da matriz $A$ prximo de $a$ ao calcular um autovalor da matriz $M = (A - aI)^{-1}$. Observe que "multiplicar" por $M$ equivalente a resolver o sistema linear com matriz $A - aI$, o que permite usar np.linalg.solve para fazer as iteraes.

Obs: note que, aps calcular um par autovetor-autovalor de $M$, ser necessrio fazer operaes simples para converter em um par autovetor-autovalor de $A$.

def any_eigen(A, a, tol=1e-6):
    """
    Encontra o par autovetor-autovalor de 'A' de autovalor mais prximo de 'a'.
    Retorna, tambm, o nmero de iteraes.
    """
    n,m = np.shape(A)
    assert n==m, 'A must be square'

    u = np.random.rand(n)
    i = np.identity(n)
    M_i = A - a*i
    ns = 0 
    xk = u/u[np.argmax(np.abs(u))]
    while True:
        ns +=1
        yk = np.linalg.solve(M_i,xk)
        b = np.dot(yk,xk)/np.dot(xk,xk)
        b = 1/b + a
        if abs(np.sum(A@xk - b*xk))<=tol:
            break    
        xk = yk/yk[np.argmax(np.abs(yk))]
    return xk, b, ns
