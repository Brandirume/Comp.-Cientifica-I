Questão 1: Todos os pontos da bisseção¶

Neste teste vamos estudar a evolução do erro no eixo $y$ no decorrer do algoritmo da bisseção. Para isso, vamos usar uma variação do método da bisseção que retorne uma lista com todos os pontos percorridos pelo algoritmo até atingir o critério de parada.

Implemente a função bissecao_pts, utilizando como critérios de parada xtol, ytol e maxiter.

def bissecao_pts(f, a, b, xtol=1e-8, ytol=1e-8, maxiter=100):
    pts=[]
    m = (a+b)/2
    fm = f(m)
    iters=1
    pts.append(m)
    while (abs(b-a) >= xtol) and (iters < maxiter) and (abs(fm) > ytol):
        if fm*f(a) < 0:
            b = m
            m = (a+b)/2
            fm = f(m)
            if a not in pts:
                pts.append(a)
            if m not in pts:
                pts.append(m)
        else:
            a = m
            m = (a+b)/2
            fm = f(m)
            if m not in pts:
                pts.append(m)
        iters+=1
    return pts
