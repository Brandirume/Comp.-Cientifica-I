Teste 5: Erros parciais e erros totais¶

Parte 1: Método do retângulo¶

Para calcular uma aproximao para uma integral, o mtodo do retngulo divide o intervalo de integrao em $N$ intervalos $I_i = [x_i,x_{i+1}]$, com $i=0,\ldots,N-1$. O erro parcial no intervalo $I_i$ definido como a diferena entre a aproximao do mtodo do retngulo e a integral exata neste intervalo.

Questão 1: Todos os termos do método do retângulo¶

  • uma lista (ou np.array) com os pontos iniciais dos intervalos utilizados; e
  • uma lista (ou np.array) com a aproximao da integral em cada intervalo.

Student's answer Score: 2.0 / 2.0 (Top)

Cada erro, que a rea da funo em que um retngulo da soma de Riemann no cobre, para cada diviso, calculado da seguinte maneira: $e_k = f(x_k)\cdot h-\int_{x_k}^{x_{k+1}}f(x)\;dx$, tal que $f(x_k)\cdot h$, a rea de um retngulo, pode ser escrito como a integral de uma constante: $\int_{x_k}^{x_{k+1}}f(x_k)\;dx$, ento a expresso toda vira $\int_{x_k}^{x_{k+1}}[f(x_k) - f(x)]\;dx$. Usando a srie de Taylor para o integrando e fazendo substituio de variveis, podemos reescrever cada erro como $e_k= -h\cdot \int_{0}^{1}[f'(x_k)\cdot th + resto(th)]\; dt = -f'(x_k)\cdot \frac{h^2}{2} + o(h^2)$. Portanto, podemos ver porque o grfico se parece com a curva de um cosseno invertido em relao ao eixo x e achatado no eixo y.

def trap_parcs(f, a, b, N):
    x0s = []
    ints = []
    xs = np.linspace(a, b, num=N, endpoint=False)
    d = (b-a)/N
    ints = (f(xs) + f(xs+d))/2* d
    return xs, ints
